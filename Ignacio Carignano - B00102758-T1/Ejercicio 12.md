# Ejercicio Nro: 12

## Enunciado
Utiliza los 7 principios del desarrollo Lean para organizar el cursado y aprobación de este año.

Qué:
Aplicar cada principio a distintas etapas del proceso de cursado.
Cómo:
Para cada principio indicar 3 ejemplos de aplicación para tu situación como estudiante.
Agregar 1 ejemplo adicional que pueda servir para estudiantes en general.
b)
Utilizar un LLM para realizar una tabla de relación de principios.

## Resolución
🔹 1. Eliminar desperdicios (Eliminate Waste)

Este principio busca reducir actividades que no aporten valor al proceso.

Aplicación personal:

  1- Evitar estudiar contenido irrelevante para parciales o trabajos prácticos.
  2- Organizar apuntes y materiales en carpetas para no perder tiempo buscando información.
  3- Reducir distracciones como redes sociales mientras estudio.
  
Ejemplo general:

Evitar reuniones grupales innecesarias que no aporten al avance del trabajo práctico.

🔹 2. Amplificar el aprendizaje (Amplify Learning)

Lean propone aprender constantemente mediante feedback y mejora continua.

Aplicación personal:

  1- Resolver ejercicios prácticos después de cada clase.
  2- Revisar errores de parciales anteriores para mejorar.
  3- Consultar dudas rápidamente en clase o con compañeros.

Ejemplo general:

Realizar simulaciones de examen para detectar debilidades antes de rendir.

🔹 3. Decidir lo más tarde posible (Decide as Late as Possible)

Consiste en evitar tomar decisiones apresuradas sin información suficiente.

Aplicación personal:

  1- Elegir tema del trabajo práctico después de entender bien la materia.
  2- No cerrar una solución técnica hasta analizar alternativas.
  3- Esperar feedback del profesor antes de rehacer completamente un trabajo.
  
Ejemplo general:

Esperar definiciones finales de una consigna antes de desarrollar todo el proyecto.

🔹 4. Entregar lo más rápido posible (Deliver as Fast as Possible)

Se busca generar avances frecuentes y funcionales.

Aplicación personal:

  1- Entregar actividades antes de la fecha límite.
  2- Dividir trabajos grandes en pequeñas tareas semanales.
  3- Avanzar continuamente en el TP en lugar de hacerlo completo al final.
  
Ejemplo general:

Presentar versiones parciales de un proyecto para obtener correcciones tempranas.

🔹 5. Empoderar al equipo (Empower the Team)

Las personas que realizan el trabajo deben tener participación y autonomía.

Aplicación personal:

  1- Participar activamente en trabajos grupales.
  2- Proponer ideas y mejoras dentro del equipo.
  3- Distribuir tareas según habilidades de cada integrante.

Ejemplo general:

Permitir que cada integrante del grupo decida cómo resolver su parte del proyecto.

🔹 6. Construir integridad (Build Integrity In)

El sistema debe ser coherente, mantenible y funcional.

Aplicación personal:

  1- Mantener ordenados apuntes y código fuente.
  2- Documentar correctamente trabajos prácticos.
  3- Revisar ejercicios antes de entregarlos.

Ejemplo general:

Mantener un formato uniforme y organizado en las entregas grupales.

🔹 7. Ver el todo (See the Whole)

Lean propone comprender el proceso completo y no solo tareas aisladas.

Aplicación personal:

  1- Organizar horarios considerando todas las materias.
  2- Relacionar conceptos entre distintas asignaturas.
  3- Planificar el cuatrimestre completo y no solo parciales individuales.

Ejemplo general:

Analizar la carga total de trabajos y exámenes antes de planificar el estudio.

## Conclusión

Los principios Lean permiten organizar el cursado de manera más eficiente, mejorando la planificación, el aprendizaje continuo, la organización personal y el trabajo en equipo. Además, ayudan a reducir desperdicios y a enfocarse en actividades que realmente aporten valor al proceso de aprendizaje.
